# Plastic-Storage
Design and calculation of Storage Tanks made up of polyOlefins for chemical industries and urban applications. 

This project is about design of Polyethylen (PE) and polypropylen tanks. These tanks can be used in applications such as storing Fresh Water, Crude Water, Wast Water, Chemicals, Acids, Bases and solvants. 
Cylandrical and Rectangular Tanks. Tanks may be design and build using only plastics or benefits form composite reinforcment.
